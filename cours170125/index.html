Cours sur les Animations CSS

Les animations CSS permettent de créer des effets de transition et de mouvement sans utiliser JavaScript.

Animation avec @keyframes

Les keyframes définissent les étapes d'une animation :

  • @keyframes : Permet de définir une séquence d'animation.
  • from et to : Définissent les états de début et de fin.

Transition avec transform

Les transitions permettent de lisser les changements de propriétés :

  • transition : Définit la durée et l'effet de transition.
  • transform : Permet d'appliquer des transformations comme scale, rotate, et translate.

Animation avec Translate

translateX et translateY permettent de déplacer un élément sur les axes X et Y.

Animation avec Scale et Rotate

scale permet d'agrandir/réduire un élément, tandis que rotate le fait pivoter.
